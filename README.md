# ActGraph

This is the repository for the project page of **ActGraph: Environment-Aware Human Action Perception from Unconstrained Videos via Semantic Scene Graphs for Robot Imitation**.

Submitted to the 10th Conference on Robot Learning (CoRL 2026).

## Overview

ActGraph is a modular perception framework that transforms raw human demonstration videos into structured, robot-executable representations — keyframes, object-centric semantic scene graphs, and semantic action sequences — while maintaining cross-scene object identity consistency.

**Key results:**
- 84.9% Keyframe F1 on Assembly101
- 95.2% Cross-scene object matching accuracy
- 91.4% Verb accuracy / 76.8% Full-tuple accuracy on Assembly101
- 100% task completion in real-robot trials
