Abstract

Understanding manipulation skills from unstructured videos is a fundamental challenge for enabling autonomous system design. Existing approaches rely on low-level visual features or language descriptions, often lacking temporal consistency and robustness to viewpoint changes.

We propose a semantic-consistent perception framework that transforms raw human videos into structured representations: keyframes, object-centric scene graphs, and semantic action sequences. The framework maintains object identities consistently under camera motion and scene transitions. Its scene graphs evolve over time and explicitly encode spatial and interaction relationships, enabling robust skill extraction.

We evaluated the method on 5 complex, multi-view, unconstrained manipulation recordings; the structured outputs supported downstream task understanding by generating executable manipulation behaviors consistent with the original human videos.

Contributions

Industrial video event understanding — A rule-based keyframe selector over VLM-derived frame attributes reliably detects manipulation events despite camera and appearance changes, achieving an 84.9% F1 score for keyframe recognition on Assembly101.
Cross-scene object grounding — A confidence-gated hybrid of visual and VLM-based matching preserves object identity across scene boundaries with 95.2% accuracy.
SSG-grounded chain-of-thought (SSG-CoT) action extraction — Conditions VLM reasoning on scene-graph structure for coherent action sequences, improving action understanding quality with 91.4% verb accuracy and 76.8% full-tuple accuracy on Assembly101.
Multi-scene manipulation video dataset — Full SSG, keyframe, and action annotations for systematic evaluation of cross-scene perception across 5 industrial demonstration videos.

Method

ActGraph is a modular four-stage pipeline for extracting grounded action sequences from human demonstration videos.

1
Scene Change Detection

Histogram-based Bhattacharyya distance partitions the video into M visually contiguous scenes. Peaks above threshold τscene = 0.35 mark scene boundaries, preventing cross-scene identity fragmentation.

2
VLM-Based Keyframe Extraction

Frames sampled at 5 FPS are encoded with a Swin Transformer. Qwen2.5-VL is queried with a structured prompt; cross-modal attention identifies action, hand state, and object-contact changes to select semantically significant keyframes.

3
SSG-Based Action Recognition

A Semantic Scene Graph Sk = (Ok, Rk) is built per keyframe using GroundingDINO + SAM2. Structural deltas ΔSk between consecutive graphs drive verb-inference rules and LLM-decoded action tuples ⟨verb, agent, patient, target, tool⟩.

4
Cross-Scene Object Matching

A confidence-gated tiered strategy combines 256-D ReID visual embeddings with class, temporal, and context bonuses. Ambiguous matches invoke VLM semantic verification, populating a global registry G with consistent object identities across all scenes.

End-to-End System Validation

We validate ActGraph end-to-end from video perception through simulation to real-world execution. Extracted action sequences are translated into a simulation with a Franka robot and executed using predefined primitives (PICK, PLACE, PRESS). On a physical Franka Research 3 arm replicating the pick-and-place + E-stop scenario, the robot successfully executed the full action sequence. Closed-loop validation — comparing the semantic action sequence extracted from the robot execution against the original human demonstration — achieved an 87% match, demonstrating successful preservation of manipulation semantics across human-to-robot transfer.
